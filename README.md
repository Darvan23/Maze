# Forest Maze Escape

A 15-chapter browser maze game built with HTML, CSS, and JavaScript.

The player controls Aimee, a little girl carrying an old lantern through huge procedural forest mazes while wolf packs chase her. Reach the ranger cabin to finish each chapter and continue the story.

## How to play

- Open `index.html` in a browser.
- Before starting, choose sound on/off and pick Aimee, Hannah, Sophie, Maddy, Akhink, Rojin, or Hajin as the hero skin.
- On PC, use arrow keys or WASD.
- Use Space or F to use Aimee's current tool.
- On phone, use the on-screen buttons, the tool button, or swipe on the maze.
- Each level has a different tool: bones, slingshot stones, flares, snares, whistles, or smoke.
- Meet Akhink, Darvan, Zach, Rojvan, Aryan, and Leslie for story moments and help.
- At the end of each chapter, answer a memory quiz about what a character said to win a trophy.
- The real mazes are much larger than the visible board, so use branches, decoys, and supply bags to survive.
- Avoid fallen logs and do not let the wolves catch you.
